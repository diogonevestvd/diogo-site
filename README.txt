Como abrir

1. Extrai o ZIP.
2. Abre a pasta.
3. Dá duplo clique em index.html.

Melhor forma no VS Code
1. Abre a pasta.
2. Instala a extensão Live Server.
3. Botão direito em index.html.
4. Open with Live Server.
